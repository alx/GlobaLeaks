Tulip
=====

TULIP stands for Temporary Unique Link Information Provider. It aims to provide
a means between Targets and Wirstleblower.

Features:
  - expires after a certain amount of downloads;
  - it is unique for every target/material;
  - the data can be stored in a flexible and configurable container

Notifications:
  -  Flexible and expandible notification system (email, twitter, facebook, SCP,
     ticketing system, ...);

Privacy
-------
Anonymity:
 - TOR Hidden serces for pubblishing, protecting wirstleblower and node mainteiner.
 - TOR Client for notifications
